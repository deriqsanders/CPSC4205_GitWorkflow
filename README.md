# Student Information Card

A simple program that displays a formatted profile card with student details. 

## Features

- Display student name (original)
- Display major and graduation year (original)
- Display email address (added)
- Display hometown (added)
- Display hobby (added)

## Technologies Used

- Python 3.14+
- Git for version control
- GitHub for repository hosting

## Installation and Setup

### Prerequisites

Latest version of Python and a Github account

### Steps to Run

1. Clone this repository:
```
Git clone https://github.com/deriqsanders/CPSC4205_GitWorkflow.git
```

2. Navigate to the project directory:
```
cd CPSC4205_GitWorkflow
```

3. Run the program:
```
python student_info.py
```

## Usage

Simply run the program to see the formatted student information card.
